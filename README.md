# Sensor-Based Smart Street Lighting System Using Arduino and LDR

This repository contains my **Task 2** submission for the **CodeAlpha Internet of Things (IoT) Internship**. The project demonstrates a **sensor-based smart street lighting system** using an **Arduino Uno** and a **Light Dependent Resistor (LDR)** simulated in **Tinkercad**.

The system automatically detects ambient light intensity and controls the LEDs accordingly. During daylight, the LEDs remain **OFF** to conserve energy, while in low-light conditions, the LEDs turn **ON** in a running sequence, representing an intelligent street lighting system.

## 🚀 Features

- Automatic light detection using an LDR
- Arduino-based automation
- Energy-efficient street lighting concept
- Running LED sequence in low-light conditions
- Tinkercad simulation (No physical hardware required)

## 🛠️ Components Used

- Arduino Uno
- Photoresistor (LDR)
- LEDs
- 220Ω Resistors
- 10kΩ Resistor
- Breadboard
- Jumper Wires
- Tinkercad
- Arduino IDE

## ⚙️ Working Principle

- The LDR continuously senses the surrounding light intensity.
- The Arduino reads the sensor value through analog pin **A0**.
- If the environment is bright, all LEDs remain **OFF**.
- If the environment becomes dark, the Arduino activates the LEDs in a running sequence, simulating an automatic street lighting system.
